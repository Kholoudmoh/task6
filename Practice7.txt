Question 1:
What is the purpose of the ? operator in Dart null safety?

Answer:
The ? operator allows declaring a nullable type that can hold null.

Example:
int? age = null;

Question 3:
How do you declare a nullable type in Dart null safety?

Answer:
By adding ? after the type.
Example:
String? city;

Question 5:
Write a function that accepts a nullable int parameter and returns 0 if the value is null using null coalescing operator ??.

Code:
int checkValue(int? value) {
  return value ?? 0;
}

void main() {
  int? number = null;
  print(checkValue(number)); // Output: 0
}

--------------------------------------------
